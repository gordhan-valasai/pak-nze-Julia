# PAK-NZE-Julia Canonical Unit System (v0.4.0)

This document is the unit contract for model code. Model equations must use canonical units; conversions to reporting units happen once at extraction/output.

- Capacity stock variables: **MW**; report as **GW**.
- Power variables: **MW**.
- Slice-level energy variables: **MWh per slice**.
- Annual energy variables: **MWh per year**; report annual generation as **TWh**.
- Primary fuel-use variables: **MWh per year**; report fuel totals as **PJ**.
- Emissions variables: **tonnes CO2-eq per year**; report as **Mt CO2-eq**.
- CAPEX coefficients: **USD per MW** (loaded as converted values if source is USD/kW).
- Fixed O&M coefficients: **USD per MW-year**.
- Variable O&M coefficients: **USD per MWh**.
- Fuel-price coefficients: **USD per MWh fuel input**.
- Carbon-price coefficients: **USD per tonne CO2-eq**.
- Objective value: **USD**; report total cost as **USD billion**.
- Time sets: integer **year**, representative **slice** index.

Implementation rule:
- Do not apply hidden or repeated conversion factors in constraints/objective.
- Perform reporting conversion exactly once in `src/solution_container.jl` or CSV write paths.
